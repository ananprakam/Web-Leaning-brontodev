SYSTEM REQUIREMENTS

MINIMUM:

ต้องการโปรเซสเซอร์และระบบปฏิบัติการแบบ 64 บิต
ระบบปฏิบัติการ: Windows 10 (20H1 version or newer, 64-bit versions)
โปรเซสเซอร์: AMD Ryzen 3 1200 @ 3. GHz or Intel Core i5-4460 @ 3.1 GHz (AVX, AVX2 and SSE 4.2 support required)
หน่วยความจำ: แรม 8 GB
กราฟิกส์: AMD RX 460 (4 GB) or NVIDIA GeForce GTX 960 (4 GB)
DirectX: เวอร์ชัน 12
พื้นที่จัดเก็บข้อมูล: พื้นที่ว่างที่พร้อมใช้งาน 170 GB
หมายเหตุเพิ่มเติม: Please note that the minimum requirement for running the HD Texture Pack for Far Cry 6 is 12GB of VRAM. For 4K configurations, you will need 16GB of VRAM. If you download and run the HD Texture Pack with lower VRAM capabilities, you will encounter performance issues while playing.

RECOMMENDED:

ต้องการโปรเซสเซอร์และระบบปฏิบัติการแบบ 64 บิต
ระบบปฏิบัติการ: Windows 10 (20H1 version or newer, 64-bit versions)
โปรเซสเซอร์: AMD Ryzen 5 3600X @ 3.8 GHz or Intel Core i7-7700 @ 3.6 GHz (AVX, AVX2 and SSE 4.2 support required)
หน่วยความจำ: แรม 16 GB
กราฟิกส์: AMD RX Vega 64 (8 GB) or NVIDIA GeForce GTX 1080 (8 GB)
DirectX: เวอร์ชัน 12
พื้นที่จัดเก็บข้อมูล: พื้นที่ว่างที่พร้อมใช้งาน 170 GB
หมายเหตุเพิ่มเติม: Please note that the minimum requirement for running the HD Texture Pack for Far Cry 6 is 12GB of VRAM. For 4K configurations, you will need 16GB of VRAM. If you download and run the HD Texture Pack with lower VRAM capabilities, you will encounter performance issues while playing.
